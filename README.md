# Robot Testing Framework

### Overview
This automated testing framework is designed for [*FIRST*Â® Robotics Competition](https://firstinspires.org/) robots programmed in Java. It allows you to define a series of tests programatically, run them sequentially, and then display results in a web dashboard. It can be adapted for any other platform as long as the dependencies are included properly.

### Requirements
Robot Side
- NetworkTables

Client Side
- Node.js (for development)

### Installation
- NetworkTables (WPILib)

### Building
Open the `/app` folder in the Terminal and run `npm install`.

### Running
Open the `/app` folder in the Terminal and run `npm start`.

### Other
Uses the Electron framework and `wpilib-nt-client`.
