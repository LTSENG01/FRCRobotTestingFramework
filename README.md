# FRC Robot Testing Framework

### Overview
This testing framework is designed for FIRSTÂ® Robotics Competition robots programmed in Java. It can be adapted for any other platform as long as the dependencies are included properly.

### Requirements
Robot Side
- WPILib
- NetworkTables

Client Side
- Node.js (for development)

### Installation
- NetworkTables (WPILib)

### Building
Open the `/app` folder in the Terminal and run `npm install`.

### Running
Open the `/app` folder in the Terminal and run `npm start`.

### Other
Uses the Electron framework and `wpilib-nt-client`.
